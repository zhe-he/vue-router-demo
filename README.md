# vue-router-demo   

## install 	
`npm install` 

### start     
`npm start`     

### build     
`npm run build` 	
 
### pack    
`npm run pack` 

### clean   
`npm run clean`     

### test 
`npm test`

### 项目介绍    
1. src/api/ 接口api      
1. src/libs/ 插件目录   
1. src/modules/ 自定义模块目录 
1. src/store/ vuex目录   
1. src/styles/ css目录    
1. src/view/ 页面目录   
1. src/router vue-router路由文件    
1. static/ 静态资源文件        
1. dist/ 输出/上线 目录，所有被编译的文件都会输出到此目录
